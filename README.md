初赛最终排名：7/2972

Accuracy:    94.0%    and    59.42%

实验组成部分：

	在原始代码的基础上尝试样本不平衡采样，重新调整batchsize，调整输入图片大小，kv，focalloss等。

	最终解决方案：

		batchsize设为12，图片resize到550之后再随机裁成512
		测试时，将图片随机randomcrop 30次，然后投票产生最终的分类结果。
